◆ s21_create_matrix()

int s21_create_matrix ( int rows,
int cols,
matrix_t * result )

Эта функция создает матрицу заданного размера, выделяет под нее память с использованием calloc.

Для рядов выделяем память: count_of_vertexes + 1. Для колонок выделяем память: 3, так как у нас всего три координаты, которые мы хотим хранить.

Parameters
rowsКоличество рядов (строк) в матрице.
colsКоличество столбцов (колонок) в матрице.
resultУказатель на указатель на матрицу (тип matrix_t). Функция выделит память и сохранит указатель на созданную матрицу в этой переменной.
Returns
0, если создание матрицы прошло успешно, 1 в случае ошибки (например, неверные аргументы).
